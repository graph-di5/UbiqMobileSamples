=============================================================================================
Getting started
=============================================================================================
Follow 5 simple steps:

1. Build a new solution from the template. If successful, new C# projects should 
   be automatically generated (one project per diagram component) 
2. Add your C# code in the automatically generated sources
3. If needed, create your UI from .ubiqml files.
4. When ready, run your application in emulator mode using one of the following methods:
	- Press F5 key.
	- On the menu bar, choose Debug, Start Debugging.
	- On the toolbar, choose Start Debugging button.
5. For access from a mobile device, deploy your application onto the cloud (choose Ubiq 
   Deploy item in the right-click context menu in Solution Explorer window)
   See details in http://ubiqmobile.com/en/startdevelop.html#runapp 
   (Run application on your mobile device)

GeneratedCode.tt
